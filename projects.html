Rendering with my WebGPU graphics library

GFX-MAIN

Gfx-Main is a high-level graphics library that I am developing for the web. It is written in TypeScript and uses WebGPU for real-time rendering. It currently supports basic 3D rendering, lighting, materials, textures, and .obj file loading. It is still in development, and I am planning to add more features like integrating real-time ray-tracing, similar to the one in the home page of this website.

Skills: TypeScript, JavaScript, WebGPU, WGSL

3D Poisson Equation Solver

This is a 3D Poisson equation solver that I developed as a final project for my graduate-level parallel computing course. It uses the Jacobi iterative method and the Gauss-Seidel Red-Black method to solve the Poisson equation in 3D. It is written in C++ and uses OpenMP, MPI, and CUDA for parallelization on CPU and GPU.

Skills: C++, OpenMP, MPI, CUDA

3D Gaussian Splatting Camera Optimization

For this project, we built on the 3D Gaussian Splatting for Real-Time Radiance Field Rendering research paper Kerbl et al. (2023) and used AI to do camera optimization to find the most memorable camera position in the 3D space. We used PyTorch to do training on the differentiable renderer and find the optimal camera position based on the ResMem score achieved by the image rendered from that camera position.

Skills: Python, PyTorch
